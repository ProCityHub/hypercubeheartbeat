# LL-07 Solo Sandbox

## Status

This is a sandbox document.

It does not collect real LL-07 rating data.

It does not authorize the empirical LL-07 run.

It does not create an outcome.

It does not upgrade any claim.

## Why this exists

The LL-07 preregistered study requires independent human raters.

The minimum valid rater count remains:

N = 12 valid independent raters

If independent raters are not available, the system must not pretend otherwise.

The correct solo path is to build and test the pipeline without treating the output as evidence.

## Allowed solo work

A single operator may:

- inspect the rating template
- test whether files open correctly
- generate synthetic demo ratings
- run analyzer tests on synthetic demo data
- improve instructions for clarity in future PRs
- prepare future intake structure
- write notes about usability problems

## Forbidden solo work

A single operator may not:

- count as multiple raters
- duplicate themselves into artificial raters
- call self-ratings an empirical LL-07 result
- enter self-ratings into the outcome log
- use synthetic ratings as evidence
- upgrade the theory based on sandbox output
- replace the preregistered N = 12 rule
- edit the frozen word list after seeing any ratings
- edit the frozen analysis script after seeing any ratings
- touch Kubota holdout data
- score LL-06 material

## Synthetic demo data

Synthetic demo data may be generated only to test the pipeline.

Synthetic demo data is not human data.

Synthetic demo data is not evidence.

Synthetic demo data may not appear in the outcome log.

Synthetic demo data may not be used to decide whether LL-07 succeeded.

## Personal self-rating

The operator may privately fill out the rating sheet for learning or usability testing.

If this happens, it must be labeled:

personal sandbox only

Personal self-rating does not count toward the preregistered rater count.

Personal self-rating is not part of LL-07 evidence.

## Standing boundary

The lab is allowed to keep building alone.

The lab is not allowed to fake independence.

The honest sentence is:

LL-07 is ready for real independent raters, but the single-operator pathway remains sandbox-only until those raters exist.
