Adds DIRECTIVE-008O: C_star Record Schema and Claim Maturity Viewer Alignment.

This replaces blunt forbidden-claim record language with evidence-based claim maturity language.

Core law: nothing is forbidden to think. Only unsupported truth presentation is blocked.

The triadic deep-question record now uses not_claimable_yet by default and supports evidence-based statuses such as raw_speculation, mathematical_candidate, testable_hypothesis, provisional_operational_claim, and rejected.

It adds an observation bank where unproven ideas can be redefined, challenged, observed later, or matured with evidence without becoming runtime memory, proof, or public claim.

The cognitive-cycle viewer now accepts newer cycles with evaluation and standing_boundary fields instead of failing only because evolution_contract is absent.

This PR creates no autonomous action, no automatic memory append, no experiment execution, no lab result, no network call, no LLM call, no public claim, and no claim upgrade without evidence.
