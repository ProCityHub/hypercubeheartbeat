# App Runtime

## Purpose

This folder will hold future runtime plans for app-facing GARVIS components.

Current status: skeleton only.

No runtime service is active yet.

Future runtime work may define:

- local-only process model
- Termux bridge process
- app shell launch path
- foreground-only mode
- approved background service mode
- local SQLite access pattern
- local log handling

Runtime data must not be committed.

Background operation requires a separate directive.
